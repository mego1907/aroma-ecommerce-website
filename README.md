## Ecommerce project using modern technologies:-

---

- ReactJS
- React Hooks
- Redux
- Redux Persist
- react-router-dom
- react-slick
- JSX
- react-icons

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```

To create a production build:

```sh
npm run build
```
